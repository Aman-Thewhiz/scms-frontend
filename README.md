# 🎓 Smart Campus Management System (Frontend)

A modern web-based system to manage campus operations including students, faculty, courses, and attendance.

---

## 🚀 Features

- 🔐 Role-based Login (Student / Faculty / Admin)
- 📊 Dashboard Interface
- 📚 Course Management UI
- 📝 Attendance Tracking UI
- 🏫 Department & Facility Pages
- ⚡ Fast and responsive UI

---

## 🛠 Tech Stack

- React (Vite)
- Chakra UI
- Axios
- React Router
